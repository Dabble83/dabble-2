# Design Acceptance Checklist

Compare the current implementation at `/design/reference` with the reference image at `/concepts/dabble-landing-concept.png`. Each item must match the reference design.

## Layout & Structure
- [x] Navigation bar alignment: Logo left, links right, proper spacing and padding
- [ ] Hero section layout: Headline and subtext positioning, CTA button placement and spacing
- [ ] 4-step section: Grid layout, card spacing, icon-to-text spacing, equal card heights
- [ ] Email capture section: Centering, form field spacing, button alignment
- [ ] Footer: Layout, spacing between elements, alignment

## Typography
- [ ] Heading hierarchy: Font sizes, weights, and line heights match reference
- [ ] Body text styling: Font size, color, and line spacing for subtext and descriptions
- [ ] Button text: Font size, weight, and letter spacing
- [ ] Navigation links: Font size, weight, and spacing

## Colors & Background
- [ ] Background watercolor effect: Gradient colors, opacity, and layering match reference
- [ ] Paper texture overlay: Subtlety and opacity of texture pattern
- [ ] Card backgrounds: White/transparent backgrounds, border colors and opacity
- [ ] Text colors: Primary text, secondary text, and link colors

## Spacing & Sizing
- [x] Section padding: Top and bottom padding for each major section
- [ ] Container max-width: Content width constraints match reference
- [ ] Card padding: Internal spacing within step cards and category cards
- [ ] Icon sizes: SVG icon dimensions relative to text

## Components & Styling
- [x] CTA button: Background color, border, padding, corner radius, shadow
- [ ] Step cards: Border style, background opacity, hover states
- [ ] Scenic illustration: SVG shapes, colors, opacity, positioning
- [ ] Form inputs: Border style, padding, focus states

## Visual Details
- [ ] Logo styling: Font family, size, color, any special effects
- [ ] Icon illustrations: Style consistency, color palette, detail level
- [ ] Border radius: Corner rounding on cards, buttons, inputs
- [ ] Shadows: Box shadows on cards, buttons, or other elements
