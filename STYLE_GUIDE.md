# Dabble Style Guide

**Quick reference for UI changes. See `spec/DABBLE_DESIGN.md` for complete design system.**

## 10 Core Design Rules

1. **Minimal, Editorial, Warm, Calm** - Design should feel like a well-designed magazine, not a web app. Text is primary. Generous whitespace. No decorative elements or visual noise.

2. **Typography: Serif for Content, System Sans for UI** - Use serif (Georgia) for headlines and body text. Use system sans (`-apple-system`) for labels, forms, and navigation. Max line length: 65-75 characters.

3. **Color Palette: Warm Neutrals + Muted Sage Green** - Background: `#f6f1e8` (warm off-white). Accent: `#7A8F6A` (muted sage green). No gradients, no bright colors, low contrast. Text: `#1F2A37` primary, `#4B5563` secondary, `#6B7280` tertiary.

4. **Spacing: 4px Base Unit, Generous Whitespace** - Use 4px multiples (8px, 16px, 24px, 32px, 48px, 64px). Prefer more space over less. Padding inside cards: 24px minimum. Section gaps: 48px.

5. **Buttons: Flat, No Shadows, Muted Colors** - Primary: `#7A8A6A` background, `#5F6B55` border, `#F7F6F2` text. Secondary: `#F3F4F6` background, `#D1D5DB` border. No gradients, subtle hover (200ms transition).

6. **Cards: White, Subtle Border, No Shadows** - Background: `#ffffff`. Border: `1px solid #e5e7eb`. Border radius: `12px`. Padding: 24px minimum. No box shadows.

7. **Forms: Clean Inputs, Clear Labels** - Input: `#ffffff` background, `2px solid #D1D5DB` border, `12px 16px` padding. Focus: border `#7A8A6A`, outline `2px solid rgba(122, 143, 106, 0.2)`. Labels: `14px`, weight `500`, `8px` margin below.

8. **Links: No Underlines, Color Transitions** - Inline links: same color as body text, hover to `#111827`. No underline by default. Navigation links: `14px`, weight `500`. Active state: `#7A8A6A`.

9. **Language: No Marketplace/Hustle Words** - Use: "share," "learn," "connect," "exchange," "discover," "dabblers." Avoid: "buy," "sell," "marketplace," "vendor," "customer," "scale," "optimize," "hustle," "transaction."

10. **Motion: Subtle Only, User-Initiated** - Transitions: max 200ms, ease-in-out. No auto-play animations. Respect `prefers-reduced-motion`. Hover/focus states only. No parallax or flashy effects.
